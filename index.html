Event Management System Mobile Project description

Layermarkâ€™s Event Management System (EMS) is an existing product that enables organizations such as DC Water to tackle the issue of severe weather conditions and ageing infrastructure.

I was tasked with implementing the modile version of the web-application for use by high level executives with the advantage of accesibility "on the go"

The app offers 2 dashboards: FloodTrack and WaterTrack. It allows DC Water to monitor and respond to an emergency faster by using data from multiple real-time feeds. This helps to identify the location and severity of a problem and determine which crews to dispatch to which locations.

The dashboards include:

  • Geo map with markers for reported incidents and work orders
  • Alert level visualization through border color and incident count boxes per map area
  • Incident/Work order details card onclick of the marker
  • Incidents/Work orders list with filtering and search capabilities

Additionally, the app utilizes user management and authorization. The data is fetched from the backend services to be represented on the dashboards live time.
